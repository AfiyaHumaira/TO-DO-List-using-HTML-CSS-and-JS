# 📝 To-Do List using HTML, CSS, and JavaScript

This project is a simple and interactive To-Do List web application developed using HTML, CSS, and JavaScript. It allows users to efficiently manage daily tasks by adding new tasks, marking them as completed, and deleting them when no longer needed. The application features a clean and user-friendly interface, supports real-time task updates, and uses browser localStorage to persist data so that tasks remain saved even after refreshing the page. This project demonstrates fundamental web development concepts such as DOM manipulation, event handling, and client-side storage, making it ideal for beginners and academic projects.

---

## 🚀 Features

- Add new tasks  
- Mark tasks as complete/incomplete  
- Delete tasks  
- Dynamic UI updates  
- Data persistence using localStorage  
- Simple and responsive design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
